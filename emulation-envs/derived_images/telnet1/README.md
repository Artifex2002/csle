# Telnet1

Telnet server that can be used to generate containers vulnerable to brute-force attacks

## Useful commands

```bash
make all # Deletes the current image and re-builds it
make build # Builds the image
make rm-image # Deletes the image   
```

## Services

- Flask server
- Telnet
- SSH
